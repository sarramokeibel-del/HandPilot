# HandPilot ✋🤖
HandPilot is a smart gesture-controlled robotic vehicle that enables real-time wireless control using ESP32, MPU6050, and ESP-NOW.

## Features
- Gesture-based control
- Wireless communication using ESP-NOW
- Real-time response
- Low latency
- Easy to expand
- Built with ESP32

## Hardware
- ESP32 ×2
- MPU6050
- Boost Converter MT3608
- L298N Motor Driver
- 4 DC Motors
- 4 Wheels
- 18650 Batteries
- Robot Chassis

## Software
- Arduino IDE
- ESP-NOW
- Wire Library
- Adafruit MPU6050

## System Architecture
Hand --> MPU6050 --> ESP32 (Glove) --> ESP-NOW --> ESP32 (Robot) --> Motor Driver --> Robot Moves
